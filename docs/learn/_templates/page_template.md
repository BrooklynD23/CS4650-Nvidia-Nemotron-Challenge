# Learn Page Template

## Why This Page Exists

Use this template when adding a new page to `docs/learn/`.

## Suggested Frontmatter

```yaml
---
title: Example Title
audience: beginner
page_type: concept
status: conceptual
last_reviewed: 2026-04-21
repo_sources:
  - docs/architecture/ARCHITECTURE.md
external_sources:
  - https://example.com
---
```

## Suggested Body Shape

1. `## Why This Page Exists`
2. the main explanation
3. `## Current Repo Evidence` when the page is project-specific
4. `## What Exists Today`, `## What Is In Progress`, or `## What Is Planned Next`
   when the page is a status or roadmap page
5. `## Sources`
